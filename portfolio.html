Bamazon

The Bamazon app connects a node js file with a local mySQL database to provide a virtual storefront in the command line and track stock of simulated products. Users can 'purchace' any number of items from a list by their product number, and the cost is calculated and displayed.

Burger

Burger is a fullstack app that stores information on the backend in a mysql database and can be modified by users on the frontend. Users can add new burgers to the menu and then move them over to the ‘devoured’ state, and this information will be persistent across sessions.

Flashcard Generator

Flashcard-Generator is a nodeJS application that uses the Inquirer npm package to present the user with flashcards. All flashcards are hardcoded and can be added to by modifying the flashcard.js file. At the start, the user can select whether they want to use a ‘basic’ flashcard or a ‘cloze’ flashcard.

Liri Node App

Liri-node-app is a nodeJS application that lets users search for movies, songs, or their tweets. It uses Twitter’s API, Spotify’s API, and the OMDB api and displays the results in the command line. NPM packages for Twitter and Spotify are installed to allow Node to access the respective APIs.

Giftastic

giftastic is a jQuery application that uses the giphy API. The user can click on a button to see gifs in a given topic, or type in a new search term to add a button with that topic. Once clicked, the top 10 gifs of that topic are displayed with the still image of their first frame. Once an image is clicked, the image switches from the static image to the moving gif, and it is stopped when the image is clicked again.
